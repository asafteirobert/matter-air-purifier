# Hardware

- **3D model** can be seen [here](https://a360.co/4dngQW1). Free license does not allow me to export it.
- **[Laser cut](Laser%20cut/)** — DXF files to laser cut the 2.5mm MDF boards making most of the body. Using a different thickness means some dimensions will change.
- **[Schematic](Schematic/)** — Schematic, PCB and EasyEDA Pro project. The PCB was manufactured at home with only the top side, so some connections use wires. This is why some of the routing, large vias and DRC rules look strange.
- **[Front Panel](Front%20Panel/)** — 3D model to be 3D printed, that slots in the front panel. It has slots for a microswitch and a common 128x32 OLED display.
- **[Pictures](Pictures/)** — Photos of the hardware.
